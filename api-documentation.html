QRIS Dynamic Generator API

This API allows you to programmatically generate dynamic QRIS codes, parse QRIS strings from images, and check server health. The base URL for all API requests is your server's root domain, or http://localhost:3001 if running locally.

GET /api/health

Checks the health status of the API server.

Response

{
  "status": "OK",
  "timestamp": "2026-06-14T21:00:00.000Z"
}

POST /api/generate

Generates a dynamic QRIS string and QR code image by injecting an amount into a static QRIS string.

Request Headers

  • Content-Type: application/json

Request Body

{
  "staticQris": "000201010211...", // Required: The static QRIS string
  "amount": 10000,               // Required: The transaction amount (number)
  "fee": 0                       // Optional: Service fee (number, defaults to 0)
}

Response

{
  "dynamicQris": "000201010212...", // The generated dynamic QRIS string
  "qrImage": "data:image/png;base64,iVBORw0K..." // Base64 encoded PNG image
}

POST /api/parse-image

Extracts the QRIS string from an uploaded QR code image.

Request Headers

  • Content-Type: multipart/form-data

Request Body

Send form data with a single field named file containing the image file (JPG, PNG, etc.).

Response

{
  "qris": "000201010211..." // The parsed QRIS string
}

POST /api/parse-image-url

Extracts the QRIS string from an image hosted at a given URL.

Request Headers

  • Content-Type: application/json

Request Body

{
  "imageUrl": "https://example.com/path/to/qr-image.jpg" // Required: URL of the QR image
}

Response

{
  "qris": "000201010211..." // The parsed QRIS string
}
